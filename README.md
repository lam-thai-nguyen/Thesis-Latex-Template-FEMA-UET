## Giới thiệu

Repository này cung cấp một mẫu đồ án tốt nghiệp (ĐATN) LaTeX ban đầu được xây dựng cho
FEMA (VNU-UET). Mẫu được thiết kế để có thể dễ dàng tùy chỉnh và áp dụng
cho các khoa khác trong VNU-UET.

## Đối tượng sử dụng

- Sinh viên FEMA tại VNU-UET
- Sinh viên các khoa khác sử dụng LaTeX để viết ĐATN
- Bất kỳ ai muốn một mẫu ĐATN gọn gàng, có cấu trúc rõ ràng

## Tính năng

- ĐATN
- Quyển tóm tắt

## Bắt đầu nhanh

1. Sao chép (git clone) hoặc tải xuống Repository này
2. Mở `thesis.tex` hoặc `summary.tex` bằng VS Code  
   (tham khảo [hướng dẫn này](https://mathjiajia.github.io/vscode-and-latex/) nếu bạn chưa cài đặt VS Code extension LaTeX Workshop)
3. Biên dịch bằng `lualatex`  
   (yêu cầu thêm một vài dòng cấu hình vào `settings.json`)
4. Bắt đầu viết ĐATN 👍

## Giấy phép

LPPL: Bạn được phép sử dụng và chỉnh sửa mẫu cho mục đích viết ĐATN,
cũng như điều chỉnh để phù hợp với các khoa hoặc cơ sở đào tạo khác.
Các phiên bản đã chỉnh sửa phải được đổi tên để tránh nhầm lẫn với bản gốc.

## Lời cảm ơn

Dự án này được xây dựng dựa trên Repository gốc tại  
https://github.com/leduyquang753/uet-graduation-latex.

So với bản gốc, các thay đổi về mặt mã nguồn là tương đối ít.
Đóng góp chính của dự án này là bổ sung tùy chọn lớp `summary`, cho phép
tạo ra quyển tóm tắt theo yêu cầu của FEMA (VNU-UET).

Ngoài ra, các mẹo thực tế về cách viết ĐATN được 
cung cấp trong các tệp `thesis.pdf` và `summary.pdf`.

## About This Template

This repository provides a LaTeX thesis template originally built for
FEMA (VNU-UET). It is designed to be easily adapted for other faculties within VNU-UET.

## Who This Is For

- FEMA students at VNU-UET
- Students from other faculties using LaTeX for thesis writing
- Anyone who wants a clean, structured academic thesis template

## Features

- Thesis
- Summary

## Quick Start

1. Clone or download this repository
2. Open `thesis.tex` or `summary.tex` with VSCode (follow this [tutorial](https://mathjiajia.github.io/vscode-and-latex/) if you haven't installed LaTeX Workshop extension)
3. Compile with `lualatex` (this requires adding a few lines to `settings.json`)
4. Write your thesis 👍
